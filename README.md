# Issues

Public issue tracker for Perp Walk.

## Reporting a Bug

1. **Search first** — check if someone already reported it
2. **Use the template** — click "New Issue" and fill out the bug report form
3. **Be specific** — include steps to reproduce, what you expected, what happened

## What Belongs Here

- Bugs and unexpected behavior
- UI/UX issues
- Documentation errors

## What Doesn't Belong Here

- Feature requests
- Support questions (use the community channels)
- Security vulnerabilities (report privately — see below)

## Security Issues

**Do not report security vulnerabilities here.**

If you discover a security issue, please report it privately. Contact details are in our main documentation.

## Response Time

We review issues regularly but can't guarantee specific response times. Clear, well-documented issues get resolved faster.
